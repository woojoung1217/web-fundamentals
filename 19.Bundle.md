# 번들러 (Bundler)

## 정의

- 여러 파일과 모듈을 하나의 배포 가능한 번들로 묶는 도구
- 대표적인 번들러: Webpack, Rollup, Parcel

## 주요 기능

### 1. 네트워크 요청 최적화

- **목적**: HTTP 요청 수 감소
- **방법**:
  - 다수의 파일을 하나 또는 소수의 파일로 통합
  - 예: 수백 개의 JS/CSS/이미지 파일 → 소수의 번들 파일

### 2. 트랜스파일링

- **코드 최적화**

  - Dead Code Elimination
  - Tree Shaking
  - 번들 크기 감소

- **브라우저 호환성**
  - 최신 JavaScript → ES5 변환
  - 지원 기능:
    - ES6+ 모듈
    - 비동기 함수
    - 옵셔널 체이닝
  - Babel 등 트랜스파일러와 연계

## 장점

1. **성능 향상**

   - 네트워크 요청 최소화
   - 로딩 속도 개선

2. **개발 효율성**

   - 모듈화된 코드 작성 가능
   - 최신 문법 사용 가능

3. **유지보수성**
   - 코드 구조화
   - 의존성 관리 용이
